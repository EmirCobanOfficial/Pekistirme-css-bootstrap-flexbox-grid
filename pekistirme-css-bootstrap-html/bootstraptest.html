Merhaba Bootstrap!

Bu bir örnek Bootstrap paragrafıdır.




Bu kutunun içinde padding var (iç boşluk), dışı da boşluklu.

Bu kutunun üstünde 4 birim margin, solunda 3 birim padding, altında 2 birim padding var.

m: margin (dış boşluk)
p: padding (iç boşluk)

Yönler:
t: top (üst)
b: bottom (alt)
s: start (sol)
e: end (sağ)
x: yatay (sağ-sol)
y: dikey (üst-alt)
